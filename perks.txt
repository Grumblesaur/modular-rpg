Character Creation Quirks:

Sense of Direction: Replace Knowledge with Intuition for Navigate checks, but
  receive no bonus for using orienteering tools when doing so, and receive no
  penalties for doing so without any tools at all.

Silent But Deadly: Replace Presence with Vigor for Intimidate checks.

Dueling Artist: Can use Finesse instead of Vigor when wielding a Melee weapon.

Gone Ballistic: Can use Vigor instead of Finesse when wielding a Thrown weapon.

Tread Lightly: Replace Agility with Finesse when Sneaking.
